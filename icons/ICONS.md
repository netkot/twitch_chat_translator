# Требования к иконкам расширения

## Какие файлы нужны

Манифест (`manifest.json`, поля `icons` и `action.default_icon`) ссылается на три размера:

| Файл                 | Размер   | Где используется                          |
| -------------------- | -------- | ----------------------------------------- |
| `icons/icon16.png`   | 16×16    | фавикон страницы, контекстное меню        |
| `icons/icon48.png`   | 48×48    | страница `chrome://extensions`            |
| `icons/icon128.png`  | 128×128  | Chrome Web Store, установка               |

## Куда положить

В папку `icons/` — **с теми же именами**, перезаписав текущие заглушки.
Тогда менять `manifest.json` не нужно.

## Технические требования

- **Формат:** PNG с прозрачным фоном (RGBA), отдельный файл на каждый размер.
- **Квадрат**, ровно указанные пиксели (16, 48, 128), без апскейла из мелкого.
- **Без скруглений и теней** — Chrome сам обрабатывает форму; рисуй на прозрачном фоне.
- 16×16 делай **отдельно** (упрощённый рисунок: крупная деталь, толстые линии),
  а не уменьшением из 128 px — иначе будет каша.

## Рекомендации по дизайну

- Безопасное поле: отступ ~8–10 % от краёв (для 128 px — паддинг ~10–12 px),
  чтобы иконку не обрезало.
- Узнаваемый силуэт на тему «чат + перевод» (речевой пузырь со стрелками или
  буквами А→文).
- Фиолетовый акцент Twitch: `#9147ff` / `#bf94ff` (эти цвета используются в `popup.css`).
- Хороший контраст и на светлой, и на тёмной теме браузера.

## Дополнительно для магазина (не для manifest)

Грузится прямо в Chrome Web Store Developer Dashboard, в репозиторий класть не обязательно:

- **Промо-иконка стора 128×128** — это и есть `icon128.png`.
- **Скриншоты 1280×800** (или 640×400).
- **Промо-плитка 440×280** (опционально).

## После замены

Перезагрузи расширение в `chrome://extensions`.
