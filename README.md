# Twitch Chat Translator

Расширение для Chrome (Manifest V3), которое переводит сообщения чата Twitch
**в реальном времени**. Текст сообщения заменяется переводом — переведённое
сообщение визуально не отличается от обычного.

## Возможности

- Перевод новых сообщений чата на лету через `MutationObserver`.
- Замена текста сообщения переводом без каких-либо визуальных пометок.
- **Автоопределение языка чата** по входящим сообщениям; язык чата показывается
  в попапе.
- **Перевод исходящих сообщений**:
  - **Ctrl + Enter** в поле ввода — перевести текст прямо в поле (без отправки),
    проверить и отправить вручную (надёжный режим).
  - Тумблер «Авто-перевод при Enter» — переводить и сразу отправлять.
  - «Язык моих сообщений»: «Авто (язык чата)» или явный язык.
- Три сервиса перевода:
  - **Google (без ключа)** — бесплатный неофициальный endpoint, по умолчанию.
  - **Google Cloud Translation** — официальный API (нужен ключ).
  - **DeepL** — API (нужен ключ; free-ключи оканчиваются на `:fx`).
- Выбор языка перевода (по умолчанию русский).
- **Локализованный интерфейс** на 12 языках (en, ru, uk, es, de, fr, pt, pl,
  tr, zh, ja, ko; основной/фолбэк — английский). По умолчанию язык интерфейса
  выбирается по языку браузера, но его можно переключить вручную в попапе.
- **Локализованные название и описание в магазине** (`_locales/` + `chrome.i18n`)
  на те же 12 языков — отображаются по языку браузера в Chrome Web Store и в
  `chrome://extensions`.
- Кеш переводов (LRU) — повторяющиеся сообщения не переводятся заново.
- Ограничение параллельных запросов, корректная работа при смене канала (SPA).
- **Совместимость с 7TV**: поддерживается как нативный чат Twitch, так и
  «new chat» расширения 7TV. Заменяется только текст; эмоуты, упоминания и
  ссылки 7TV остаются нетронутыми.

## Установка (режим разработчика)

1. Откройте `chrome://extensions`.
2. Включите **Режим разработчика** (Developer mode) в правом верхнем углу.
3. Нажмите **Загрузить распакованное расширение** (Load unpacked).
4. Выберите папку `C:\Work\twitch_translator`.
5. Откройте любой канал на `https://www.twitch.tv/` — сообщения чата начнут
   переводиться. Настройки — по клику на иконку расширения.

## Настройки (popup)

- **Язык интерфейса** — «Авто (браузер)» или явный выбор из 10 языков.
- **Включить перевод** — глобальный тумблер.
- **Язык перевода** — целевой язык.
- **Сервис** — выбор провайдера. Для Google Cloud / DeepL появится поле для
  API-ключа (хранится в `chrome.storage.sync`).

## Структура

| Файл            | Назначение                                                   |
| --------------- | ------------------------------------------------------------ |
| `manifest.json` | Манифест MV3, разрешения, регистрация скриптов.              |
| `background.js` | Service worker: запросы к API перевода + LRU-кеш.            |
| `content.js`    | Наблюдение за чатом и замена текста сообщений.               |
| `content.css`   | Подсветка переведённых сообщений.                            |
| `popup.html/js/css` | Окно настроек.                                          |
| `i18n.js`       | Словарь локализации интерфейса (12 языков) и хелперы.        |
| `_locales/`     | Локализация названия и описания для магазина (`chrome.i18n`).|
| `icons/`        | Иконки 16/48/128 px.                                         |

## Примечания

- Бесплатный endpoint Google не имеет гарантий и может ограничивать частоту
  запросов при очень активном чате. Для стабильности используйте ключ DeepL
  или Google Cloud.
- Селекторы DOM Twitch и 7TV иногда меняются. Если перевод перестал работать,
  обновите селекторы в `content.js` (`MESSAGE_SELECTOR`, `BODY_SELECTOR`,
  `TEXT_FRAGMENT_SELECTOR`, `CONTAINER_SELECTORS`). Текущие 7TV-классы:
  `.seventv-chat-list`, `.seventv-user-message`, `.seventv-chat-message-body`,
  `.text-token`.
